# CryptoPrime Website

A high-performance cryptocurrency trading platform website built with HTML, Tailwind CSS, and Vanilla JavaScript.

## Features
- **Modern Glassmorphism Design**: Sleek, futuristic UI with translucent elements.
- **Responsive Layout**: Works perfectly on mobile, tablet, and desktop.
- **Multi-page Navigation**: Includes Home, Market, Features, and Contact pages.
- **Interactive Elements**: Hover effects, mobile menu, and smooth scrolling.
- **Zero Dependencies**: Uses CDN for Tailwind and FontAwesome, so no Node.js installation is required.

## How to Run
Simply open the `index.html` file in any modern web browser (Chrome, Edge, Firefox).

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- FontAwesome Icons
- Google Fonts (Outfit)
